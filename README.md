# Flappy Bird Git Workshop

Learn Git fundamentals while building a Flappy Bird game in Python!

This workshop will teach you essential Git commands through hands-on practice:
- `git clone` - Get the repository
- `git checkout -b` - Create new branches
- `git add` - Stage changes
- `git commit` - Save your work
- `git push` - Upload to remote
- `git pull` - Get updates
- `git merge` - Combine branches

Follow the instructions in `INSTRUCTIONS.md` to get started!
